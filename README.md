# Ejecutar en desarrollo

1. Clonar Repo
1. Ejecutar

```{bash}
yarn install
```

3. Asegurarse de tener instalado NestCli

```bash
npm i -g @nestjs/cli
```

4. Levantar Base de datos en mongodb

```bash
docker compose up -d

```

## Stack usado

- Nest
- MongoDB
